# @vibe-forge/server 0.10.1

发布日期：2026-04-10

## 发布范围

- 发布 `@vibe-forge/server@0.10.1`

## 主要变更

- Server 新增频道动作 token、一次性消费和签名校验链路，用于安全承载会话详情页与工具结果导出链接。
- 新增 channel action 页面与相关路由装配，频道侧可以把工具调用详情、安全导出和会话跳转能力暴露为可点击动作。
- 会话服务与权限处理同步补强了频道上下文下的动作分发与权限校验，减少无签名链接或重复消费带来的风险。

## 兼容性说明

- 对外动作链接依赖 `__VF_PROJECT_AI_SERVER_ACTION_SECRET__`；未配置时不会生成可用签名链接。
- 这次变更主要扩展频道动作能力，不影响普通本地会话启动方式。
