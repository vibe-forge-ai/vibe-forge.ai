# @vibe-forge/server 0.11.2

发布日期：2026-04-13

## 发布范围

- 发布 `@vibe-forge/server@0.11.2`

## 主要变更

- `/session search` 不再要求必须传搜索词；默认会列出最近会话，方便在 Lark 中直接查看可接力的上下文。
- 新增 `/session list` 作为显式列表入口，并为列表和搜索结果统一补齐分页能力。
- Lark 私聊中的会话列表与搜索结果现在会提供上一页 / 下一页 follow-up，避免手动重复输入分页命令。
- 命令帮助中 variadic 参数的用法展示修正为可选形式，避免把“零个或多个参数”错误显示成必填。

## 兼容性说明

- 本次为向后兼容的 patch 发布，原有 `/session search <query>` 用法继续可用。
- 新增的 `/session list` 与 `--page` 分页参数属于增量能力，不影响现有绑定和恢复逻辑。
