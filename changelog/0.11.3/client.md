# @vibe-forge/client 0.11.3

发布日期：2026-04-16

## 发布范围

- 发布 `@vibe-forge/client@0.11.3`

## 主要变更

- 底部 sender 现在会把权限请求、任务进度和排队消息统一叠放在输入区上方，减少权限确认时的上下文跳转。
- 权限请求卡片支持快速选择授权选项，并在确认前阻止普通发送，避免误把权限回复当作新消息提交。
- sender 附件、adapter 选择器和提交按钮布局经过整理，长会话中的输入区状态更稳定。

## 兼容性说明

- 本次为向后兼容的 patch 发布。
- 新增的排队消息展示依赖服务端提供对应 queue API 与 websocket 事件；未升级服务端时，其他 sender 基础能力继续可用。
